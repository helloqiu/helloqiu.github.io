2018总结杂记

2019-02-02

好久没有更新 blog 了，作为一个社畜过年放假终于有时间来给已经长草的博客更新点啥了。
所以在 2019 年已经过了一个月之后，我打算来总结一下 2018 年发生的事情。

毕业

2018 年最大的事情大概就是我本科顺利毕业了。在成都待了四年，最终换来了这一纸文凭，一下子要离开它，可能也会萌生出“这辈子也再也没有一个四年来让我在这里了”的这种想法，变得怀念起来。

说来也奇怪，比起成都这座城市，学校反而让我留恋的少得多，最终想去留住的，大概只剩下了乐团和几位老师。

作为一名非主流的艺术特长生，我很庆幸大学四年有一个乐团可以让我不变得碌碌无为，能让我在一个理工科学校多少还带有几丝人文气息，多少还有半只脚点在另一个不同的世界里，而不至于固守一处，觉得世界大抵如此。

另外，我也很庆幸毕业时能在最后一个学期的毕业设计中结识荣智海老师，他不仅让我感受到了一位学者对自己专业的认真，同时也让我看到了支撑起一位师者的核心——师德。总之很感谢他在毕业设计中对我的启发式的指导和对于我申请学校的帮助，他在听说我打算申请国外学校时不仅帮我写了推荐信，还主动约我传授给了我一些经验。这些帮助让我迸发出一种“士为知己者死”的觉悟，在毕业设计中我十分努力的去做好了，可能这也是我回报老师的唯一方式了吧。

在七月十一号，我回到了青岛，我毕业了。

工作

由于雅思考试大失败，我失去了出国的机会，回到青岛之后就开始准备找工作。

在经历了一天内去北京面试再返回青岛之后，我很幸运的拿到了长亭科技的 offer，找到了一个收留我的地方。

希望我能工作的更好吧。
